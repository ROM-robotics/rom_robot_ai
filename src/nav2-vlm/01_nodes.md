# ROS 2 Navigation with VLM Nodes Explanation

ဤပရောဂျက်တွင်ပါဝင်သော Python ဖိုင်များ၏ လုပ်ဆောင်ပုံအကျဉ်းချုပ်မှာ အောက်ပါအတိုင်းဖြစ်ပါသည်။

## 1. `agent.py`
- **လုပ်ဆောင်ချက်**: ဤ script သည် AI planner အဖြစ် အလုပ်လုပ်ပါသည်။
- **အသေးစိတ်**: 
  - OpenAI ၏ GPT-4o မော်ဒယ်ကို အသုံးပြုထားသည်။
  - မြေပုံပုံရိပ် (image) နှင့် အသုံးပြုသူ၏ မေးခွန်း (ဥပမာ - "ထမင်းစားခန်းကို သွားချင်တယ်") ကို AI ထံ ပေးပို့သည်။
  - AI မှ ပြန်လည်ဖြေကြားသော လမ်းကြောင်း (coordinates) များကို ရယူပေးသည်။

## 2. `convert_image.py`
- **လုပ်ဆောင်ချက်**: ပုံရိပ်ဖိုင် Format ပြောင်းလဲပေးခြင်း။
- **အသေးစိတ်**: 
  - ROS တွင်အသုံးများသော `.pgm` မြေပုံဖိုင်များကို Vision Language Model (VLM) များ နားလည်နိုင်သော `.png` format သို့ ပြောင်းလဲပေးသည်။

## 3. `cost_map_saver.py`
- **လုပ်ဆောင်ချက်**: Costmap ကို ပုံရိပ်အဖြစ် သိမ်းဆည်းခြင်း။
- **အသေးစိတ်**:
  - `costmap_saver` ဟုခေါ်သော ROS 2 Node တစ်ခုဖြစ်သည်။
  - `/global_costmap/costmap` topic မှ data များကို ယူပြီး Robot ၏ ပတ်ဝန်းကျင် အတားအဆီးမြေပုံကို `global_costmap.png` အဖြစ် သိမ်းဆည်းပေးသည်။ ဤပုံကို `agent.py` တွင် ပြန်လည်အသုံးပြုနိုင်သည်။

## 4. `nav2_test.py`
- **လုပ်ဆောင်ချက်**: Robot Navigation ကို စမ်းသပ်မောင်းနှင်ခြင်း။
- **အသေးစိတ်**:
  - Nav2 (Navigation 2) stack နှင့် ချိတ်ဆက်အလုပ်လုပ်သည်။
  - Robot ၏ ကနဦး တည်နေရာ (Initial Pose) ကို သတ်မှတ်ပေးသည်။
  - `agent.py` မှတဆင့် AI planner ကိုခေါ်ယူပြီး ရရှိလာသော လမ်းကြောင်းအတိုင်း Robot ကို မောင်းနှင်စေမည့် အဓိက script ဖြစ်သည်။

## 5. `prompt.py`
- **လုပ်ဆောင်ချက်**: AI အတွက် ညွှန်ကြားချက်များ (Prompts) ထားရှိခြင်း။
- **အသေးစိတ်**:
  - `meta_prompt` ပါဝင်ပြီး AI ကို မြေပုံ၏ ဖွဲ့စည်းပုံ၊ အခန်းနေရာများနှင့် လမ်းကြောင်းရှာဖွေရာတွင် လိုက်နာရမည့် စည်းကမ်းများကို သတ်မှတ်ပေးထားသည်။
  - အခန်းများအချင်းချင်း ချိတ်ဆက်မှု (Adjacency List) နှင့် Pixel coordinate များ တွက်ချက်ပုံများကို လမ်းညွှန်ထားသည်။

## 6. `input_mapvlm.py`
- **လုပ်ဆောင်ချက်**: မြေပုံပုံရိပ်အား ကြည့်ရှုစစ်ဆေးခြင်း။
- **အသေးစိတ်**:
  - SLAM မှ ရရှိလာသော မြေပုံပုံရိပ်ကို `matplotlib` အသုံးပြု၍ ဖွင့်ကြည့်ရန် နှင့် စစ်ဆေးရန် အသုံးပြုသည်။

## 7. `metaprompt.py`
- **လုပ်ဆောင်ချက်**: Prompt ရေးသားနည်း လမ်းညွှန်။
- **အသေးစိတ်**:
  - OpenAI ၏ System Prompt များကို မည်သို့ စနစ်တကျ ရေးသားရမည်ဆိုသော Guidelines များ ပါဝင်သည်။ ၎င်းသည် ပရောဂျက်၏ အဓိက run-time code ထက် prompt engineering အတွက် အထောက်အကူပြုဖိုင် ဖြစ်နိုင်သည်။
